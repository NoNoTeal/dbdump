Reproduction Steps
  • (There are many ways to reproduce it, maybe occurs on older phones only)
  • Be in more than 90 quiet large servers. (Performance issue reproduce related)
  • \*Optional\* Enable notifications for all messages with mobile push enabled, in a large active chatting server
  • Freshly open Discord and choose a server where you can write
  • Type something, use "return" once see if anything happens, type after return see if it adjusts
  • If it does, use the app a bit and see if the chat field will not respond later
  • Maybe switch the channel once and type and see if it will respond or not
Expected Behavior

It should readjust the text field or reposition it. But because something else seems to be happening in the app, it is not doing that or delaying it heavily. When it does work, there's just a slight delay behind it, that's fine.

Current Behavior

During loading times or lag times due to optimization reasons, the text does not adjust itsself. When it does then its fine, but when it doesn't it is not. It's hard to say what causes it to delay it many seconds as it also occurs after it's loaded. https://www.youtube.com/watch?v=Duuxq_zuUTw
