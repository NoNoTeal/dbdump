Reproduction Steps
  • Open a server that you have Manage Channel and View Audit Log Permissions in (or create one)
  • Right click a voice channel then edit channel
  • Change user limit to any setting
  • Save Changes
  • Exit edit channel UI
  • Open Server Settings
  • Navigate to Audit Log
  • Note there is no entry for the recent channel update (even after waiting to make sure cache has refreshed)
Expected Behavior

Voice channel user limit updates should get logged in audit logs just like voice channel bitrate changes.

Current Behavior

Voice channel user limit changes do not create audit log entries.
